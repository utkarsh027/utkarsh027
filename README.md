### 📄 [PDF Q&A Chatbot](https://github.com/utkarsh027/PDF-QandA-ChatBot)
**The skill: Conversational RAG with memory**

Not just RAG — *conversational* RAG. A `history_aware_retriever` reformulates follow-up questions using chat history before retrieval, so "tell me more about that" retrieves the right chunks. Upload multiple PDFs, switch sessions, ask anything. Built on Groq's Gemma2-9b for near-instant responses.

`Python` `LangChain` `Groq` `Gemma2-9b` `ChromaDB` `HuggingFace Embeddings` `Streamlit`

### 🔍 [RAG Q&A App](https://github.com/utkarsh027/RAG_QanaA)
**The skill: Production RAG architecture**

Research paper Q&A with a clean separation of concerns: ingestion (OpenAI embeddings → FAISS), retrieval (semantic similarity search), generation (Llama3-8b via Groq). Response time measured per query. Source chunks exposed in an expandable panel. Tested on *Attention Is All You Need*.

`Python` `LangChain` `Llama3-8b` `Groq` `FAISS` `OpenAI Embeddings` `Streamlit`

---

### 🧠 [Sentiment Analysis — VADER vs RoBERTa](https://github.com/utkarsh027/Sentiment_Analysis)
**The skill: NLP model comparison + transformer inference**

Side-by-side comparison of lexicon-based VADER and transformer-based `cardiffnlp/twitter-roberta-base-sentiment` on 500 Amazon Fine Food Reviews. Ran both models on every review, merged results, and visualised divergence with a pairplot across all 6 score dimensions. Shows exactly where context beats bag-of-words.

`Python` `HuggingFace Transformers` `RoBERTa` `VADER` `NLTK` `Seaborn` `Kaggle`

---

### 🔬 [NeuroGATE — Custom Graph Transformer](https://github.com/utkarsh027/neurogate)
**The skill: Novel architecture design from scratch**

Designed and built a self-supervised graph transformer with a custom multi-head attention mechanism that injects learned edge-level bias vectors into attention scores — beyond standard Q·K similarity. Trained using BERT-style masked node reconstruction on a 33,000-cell, 99,000-feature dataset at HPC scale (SLURM). Built entirely in PyTorch + PyG during academic research at UC Irvine.

`Python` `PyTorch` `PyG` `Custom Attention` `Self-Supervised Learning` `HPC/SLURM`
